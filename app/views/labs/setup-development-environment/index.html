Software Requirements

Open network is recommended while running the Media Server Studio & OpenCV installations as corporte proxy blocks downloading certain files.

  • Install Ubuntu 16.04 by downloading 64-bit ISO from the following link: \\10.224.54.1\Raghavendra\workshop-setup\ubuntu-16.04-desktop-amd64.iso
  • Note: While installing, select the option "install Ubuntu alongside with Windows"

    • Set name: intel

    computer name: intel<NUC Number>

    Username: intel

    Password: intel123


  • Update the Ubuntu packages:
    $sudo apt-get update

    Note: Troubleshoot for core dump error on update- $sudo apt-get purge libappstream3

  • Install CVSDK, which will internally install Media Server studio also.

    Link: \\10.224.54.1\Raghavendra\workshop-setup\VAinstallerSept24_2.gz

  • Copy the VAinstallerSept24_2.gz to Documents folder and unzip the contents

    • Note: Use Ctrl+Alt+F1 to switch to one of the classic UNIX terminals and install the above SDK. It won't install properly in GUI based terminal. If you wan't to come back to GUI, press Ctrl+Alt+F7

  • Provide your credentials given during installation
  • Run the command:
    $sudo python VAinstaller.py --SDK CVSDK

    Note:- If it asks for any overwriting during installation, say 'y' - yes to confirm

  • Run the following command from VAinstallerSept24 directory:
    $./setupvars.sh
  • Reboot the system by issuing the following command:
    $reboot
  • Execute the following command from the terminal
    echo 'export PKG_CONFIG_PATH="$PKG_CONFIG_PATH:/usr/lib64/pkgconfig"' >> ~/.bashrc
  • Download opencv installation script from \\10.224.54.1\Raghavendra\workshop-setup\OpenCV to downloads directory
  • Install OpenCV by running the

    $cd '/home/intel<NUC Number>/Downloads/OpenCV/

    $bash install-opencv.sh

  • Copy the workshop zip file from \\10.224.54.1\Raghavendra\workshop-setup\Workshop.zip to /home/Documents/ and unzip it

    • Change the default password (required to proceed). The new value must be a strong password. It should contain at least one uppercase letter, one lowercase letter, one digit, and one special character, and be at least eight characters. A management console application can change the Intel AMT password without modifying the CSME password.

    If already provisioned, unprovison vPro machines as follows:

    1. Enter the MEBX ctrl-p during the boot process.

    2. Enter the vPro password that was set when the machine was configured.

    3. Select the Intel® ME General Settings

    4. Select the Unconfigure Network Acces

    5. Answer Yes to the reset network settings caution. This is just a warning that is going to reset the machine to the factory defaults.

    6. Select if it should be a Full or Partial Unprovision. Full will reset everything but the password.

    Once the machine has been unprovisioned the machine will need to have the RCFG option started.

    7. Select the Remote Setup and Configuration

    8. Select the RCFG

    9. Select Start Configuration


    Verification and Validation

    1. From the terminal Run python and test OpenCV

    $python

    $import cv2

    2. It should not give any error.

    3. Type: print cv2.__version__

    4. It should display the cv2 version

    5. Execute the following command and verify there are no errors:

    $vainfo

    6. Execute

    $cd /opt/intel/mediasdk/samples/_bin/x64

    $./sample_decode h264 -i ../content/test_stream.264 -hw -vaapi -r

    This should run a video and show the usage of hw and vaapi for decoding frames in terminal.

    5. Shutdown the system, unplug the cables and stick Intel logo provided in the box, on to the NUC and pack the box.
