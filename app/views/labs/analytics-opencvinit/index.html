Learn how to capture video from the a camera using OpenCV and Python.

  • Make sure you have connected the given Camera to you Intel® NUC
  • Mount camera at some position where you can see your face when we capture video.

To create the Python programs in this lab, we will use the terminal and gedit editor in Linux.

  1. Open terminal using Ctrl + Alt + t
  2. Now create and navigate to new directory as follows:

$cd /home/intel[workshop id]/Documents/workshop/

$sudo mkdir python_samples

$sudo chmod 777 python_samples

$cd python_samples

Create new file and save it as camera.py by following these steps:

  1. Issue the command: $sudo gedit camera.py
  2. A blank file opens up
  3. Save the file using Ctrl + s
  4. Begin by importing the numpy and opencv packages.
    5. import numpy as np import cv2
Video Capture and Display
  1. Capture the video using cv2.VideoCapture
  2. Video capture method takes one parameter. This parameter is the index number of camera that you want to use. For our workshop, we are using only one camera.. hence It will be 0.
  3. Create a namedWindow to show the output
  4. Show the output on this namedWindow using imshow method
  5. Copy the below code and put it on your file and run your application
  6. Press Esc to exit
  7. cap = cv2.VideoCapture(0) while(True): # Capture frame-by-frame ret, frame = cap.read() #Set full screen cv2.namedWindow("Target", cv2.WND_PROP_FULLSCREEN) cv2.setWindowProperty("Target",cv2.WND_PROP_FULLSCREEN,cv2.WINDOW_FULLSCREEN) # Display the resulting frame cv2.imshow('Target',frame) if cv2.waitKey(1) & 0xFF == 27: break # When everything done, release the capture cap.release() cv2.destroyAllWindows()
  8. Press CTRL + s to save the file
  9. Close the file and return to terminal. Connect the camera to the NUC and Execute:
$python camera.py
