Lab Overview
  • Here we will be build an interative face detection application which will detect a face from live camera feed.
  • We will build our face detection code from a provided main.cpp file with default header files and the classes that will be used in our next labs.

Here we will be working with TODOs for face detection only, keep all other TODOs as it is.We will be using them in our next lab

We will be replacing Face detection TODOs with the following:

  • Include the required header files.
  • Capture video frames using OpenCV API.
  • Loading inference engine plugins
  • Loading pre trained optimized face detection model
  • Request for inference on GPU
  • With the inference result, we will draw a rectangle marking the face
